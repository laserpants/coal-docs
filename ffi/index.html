C function interface

When developing applications or libraries that perform network access, disk I/O, or other tasks that interact with low-level system APIs, it is often necessary to interface with code written in C. This can be achieved in Coal through the C function interface:

  #{<c_function_name : t>}(<arg_0>, <arg_1>, ..., <arg_n>)(<continuation>)

This expression consists of three components. The first 

#{c_function_name : t}
