# `Coal.Monoid`

A monoid is a semigroup with an identity element (`id`), providing a neutral value for the associative binary operation.

---

### `Monoid`

<span class="badge badge-primary">trait</span>

Trait for types that have an associative binary operation (inherited 
from `Semigroup`) and an identity element `id` that is neutral with 
respect to that operation.

- `id : m`
